# 元素状态改变的监控方法

![](元素改变监控.png)

选项一：监听当前元素的子元素的添加、删除和修改的状态，常用于子元素的状态变化的监控上。

选项二：监听当前元素属性的变化，比如一个轮播图常常会修改当前元素的一个定位或者位置属性我们可以用它来监听。

选项三：监听当前元素的移除变化，当某个特殊阶段触发了元素的移除可以通过它进行监听。

# Console控制台

支持一些简单的快捷键:

- `Ctrl + u` 删除当前行

- `Ctrl + l` 清空控制台所有输出，不包含当前行

- `Ctrl + a` 光标移动到行首

- `Ctrl + e` 光标移动到行尾
