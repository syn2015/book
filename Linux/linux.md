# 文件和文件夹

- bin，全称binary，存储二进制文件，可运行
- dev，存放外接设备
- etc，存放配置文件，
- home，存放除root用户以外的账户信息
- proc，存放系统运行的进程
- root，root用户的信息
- sbin，全程super binary,存放有super权限的二进制文件
- tmp，存放临时文件
- usr，存放用户安装的软件
- var，存放系统/软件的日志文件
- mtn，外接设备的挂载目录
